# Simulations

Use this folder for flight-model files, assumptions, and results.

Suggested contents:

- OpenRocket, RASAero, or equivalent source files
- aerodynamic and mass assumptions
- motor files and their sources
- predicted altitude, velocity, acceleration, and stability
- sensitivity studies and launch-condition limits
- comparisons between predictions and measured flight data

Name simulation outputs with the model revision and date so results remain reproducible.
