# Flight Data

Use this folder for complete, traceable flight records.

Suggested flight folder name:

`YYYY-MM-DD_location_flight-number`

For each flight, record:

- vehicle, motor, avionics, and recovery configuration
- measured liftoff mass and center of gravity
- weather and launch conditions
- checklist and launch notes
- original telemetry and sensor files
- photos or video references
- processed analysis and comparison with simulations
- anomalies, damage, lessons learned, and next actions

Keep raw data unchanged. Store processed data and analysis separately.
