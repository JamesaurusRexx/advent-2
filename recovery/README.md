# Recovery

Use this folder for recovery-system design, deployment, packing, and inspection records.

Suggested contents:

- parachute and streamer specifications
- harness, attachment point, and protector details
- deployment architecture and event sequence
- descent-rate estimates and assumptions
- packing checklists and photographs
- ground-test plans, results, and corrective actions

Keep each flown recovery configuration traceable to a specific flight record.
