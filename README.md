# Advent 2

Advent 2 is my second model rocket I've personally developed. It is built using Onshape and a 3D printer in conjunction with Estes Rocket supplies.

## Current Status
Working on the first steps on the design. I am using OpenRocket to get a good estimate for dimensions and mass. I have decided on a E-class engine, specifically the E30-7T.
