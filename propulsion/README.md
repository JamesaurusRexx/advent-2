# Propulsion

Use this folder for certified motor selection, motor retention, vehicle integration, and motor records.

Suggested contents:

- approved motor candidates and manufacturer data
- thrust-curve source and file version
- motor mount and retention compatibility
- predicted performance for each configuration
- purchased motor inventory and lot information
- preflight and post-flight observations

Follow applicable laws and recognized safety codes. Use appropriately certified commercial motors and qualified supervision; do not treat repository notes as authorization to manufacture or modify energetic materials.
